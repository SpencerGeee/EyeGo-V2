## Deferred to production-move (out of audit scope, by decision)

These are environment/release concerns, not code defects:

1. **Secret rotation** ‚Äî the **Mapbox token** in `apps/rider/.env` is real and committed; rotate before public release. (The Paystack key is a placeholder `pk_test_REPLACE_WITH_YOUR_KEY`, not a live secret.)
2. **Postgres cutover** ‚Äî set production `DATABASE_URL`.
3. **Production API URLs** ‚Äî `EXPO_PUBLIC_API_URL` localhost values are documented dev fallbacks; supply prod URLs via EAS secrets.
4. **Store submission** ‚Äî EAS production builds + Play/App Store entitlements.
5. **Android 12+ background-location / foreground-service entitlement** ‚Äî required if continuous background tracking is needed (currently foreground watch + AppState resume).

## Backlog (Medium/Low ‚Äî documented, not fixed per decision)

- Heatmap poll continues when app backgrounded if `showHeatmap && isOnline` (battery, minor).
- Consider a `TripStatus` Prisma enum ‚Äî status is currently a free-form `String`, relying on convention across modules.
